Question 2:

Data-visualization

Before we fit and review model outputs, we should visualize our data. Review the code and plot, shown below.

In[10]:
# Exclude the 'mpg' (target variable) and non-numeric columns from the feature names
feature_names = mpg_data.select_dtypes(include=[float, int]).columns.difference(['mpg'])

# Loop through each numeric feature (column) in mpg_data
for feature in feature_names:
    # Extract the feature column and target variable (mpg)
    X_feature = mpg_data[[feature]].values  # Extract as a 2D array
    y = mpg_data['mpg'].values  # Target variable (mpg)
    
    # Create a scatter plot for the feature against the target (mpg)
    plt.figure(figsize=(6, 4))
    plt.scatter(X_feature, y, label='Data', color='blue')

    # Fit a linear regression model
    lm = LinearRegression()
    lm.fit(X_feature, y)

    # Plot the regression line
    plt.plot(X_feature, lm.predict(X_feature), color='red', label='Regression Line')

    # Add labels and title
    plt.xlabel(feature)
    plt.ylabel('Miles Per Gallon')
    plt.title(f'{feature} vs Miles Per Gallon')

    # Add a legend
    plt.legend()

    # Show the plot
    plt.show()

Answer the following questions:

(i) Describe the associations being plotted ? (i.e., positive association, negative association, no association)

Your answer here... Acceleration vs MPG: Positive association as acceleration increases, MPG tends to increase.

Cylinders vs MPG: Negative association more cylinders generally lead to lower MPG.

Displacement vs MPG: Negative association higher displacement relates to lower MPG.

Horsepower vs MPG: Negative association higher horsepower leads to lower MPG.

Model year vs MPG: Positive association newer model years tend to have better MPG.

Weight vs MPG: Strong negative association heavier cars have worse MPG.

(ii) What concept defines the plotted line?

Your answer here... The line is defined by the linear regression model (least squares fit), which represents the best linear approximation of the relationship between the predictor variable and MPG.

(iii) Do all data points in the dataset fall perfectly along the plotted line? If not, why might there be deviations between the data points and the line, and what do these deviations indicate about the relationship between the variables?

Your answer here... No, points dont fall perfectly on the line. Deviations occur due to variability, measurement error, and non-linear effects. These residuals represent the difference between actual and predicted MPG.

In[14]:
# Your code here ...
numeric_predictors = train.select_dtypes(include=[float, int]).drop(columns=['mpg'])
X_train, y_train = numeric_predictors, train['mpg']
X_test, y_test = test.select_dtypes(include=[float, int]).drop(columns=['mpg']), test['mpg']

lm = LinearRegression()
lm.fit(X_train, y_train)


# Create a DataFrame containing the slope (coefficients) and intercept
coefficients_df = pd.DataFrame({
    "predictor": X_train.columns,
    "slope": lm.coef_,
    "intercept": [lm.intercept_] * len(lm.coef_)
})
print(coefficients_df)
